# 프롬프트 시장가 검증 테스트 케이스

이 문서는 `RANKING_PROMPT`에 추가된 "시장 가격 상식 검증" 로직이 올바르게 작동하는지 확인하기 위한 기대 결과값입니다.

---

## 1. 쿼리: 아이패드 프로 M4
- **후보군 상황**: [1원(약정), 5,000원(보호필름), 1,550,000원(본체)]
- **기대 판단**:
  - 1원: 제외 (이유: 통신사 할부 미끼/상식 밖 저가)
  - 5,000원: 제외 (이유: 본품 시장가 대비 10% 미만 소모품)
  - 1,550,000원: **추천 가능** (이유: 시장 상식가 150만~260만원 범위 내)

## 2. 쿼리: 로보락 S8 MaxV Ultra
- **후보군 상황**: [1,200원(사이드브러쉬), 2,500원(더스트백), 1,750,000원(본체)]
- **기대 판단**:
  - 1,200원/2,500원: 제외 (이유: 본체 시장가 120만~200만원 대비 현저히 낮은 소모품)
  - 1,750,000원: **추천 가능**

## 3. 쿼리: 마우스패드
- **후보군 상황**: [120원(벌크 미끼), 4,500원(정품), 25,000원(게이밍)]
- **기대 판단**:
  - 120원: 제외 (이유: 옵션 미끼 또는 데이터 오류)
  - 4,500원/25,000원: **추천 가능** (이유: 마우스패드 정상 범주 3천원~5만원 내)

## 4. 쿼리: 비스포크 AI 콤보
- **후보군 상황**: [5,000원(액정필름), 2,400,000원(본체)]
- **기대 판단**:
  - 5,000원: 제외 (이유: 대형 가전 시장가 대비 소모품)
  - 2,400,000원: **추천 가능**

## 5. 쿼리: 공기청정기 필터
- **후보군 상황**: [10원(미끼), 25,000원(정품 필터)]
- **기대 판단**:
  - 10원: 제외 (이유: 미끼 가격)
  - 25,000원: **추천 가능** (이유: 본품(필터) 시장가 부합)
